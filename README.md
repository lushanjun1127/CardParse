# VCardPro - 专业VCF联系人批量生成工具

VCardPro 是一个用于批量生成VCF联系人文件的专业级静态网页应用。用户可以通过简单的文本输入，快速生成标准的VCF文件，用于批量导入联系人。

## 特性

- ✨ **简单易用**: 直接粘贴联系人信息即可生成VCF文件
- 📱 **响应式设计**: 适配桌面和移动设备
- ⚡ **原生JavaScript**: 无需安装任何依赖，纯静态文件
- 🔒 **隐私保护**: 所有数据处理均在本地完成，不会上传至服务器
- 📥 **批量处理**: 支持一次生成多个联系人的VCF文件
- 📋 **实时预览**: 解析后可预览联系人信息
- 📞 **智能解析**: 支持多种姓名格式，包括中文名、英文名和字母数字组合（如A0397）
- 📄 **VCF内容显示**: 生成的VCF内容可直接查看和复制
- 🔄 **双向转换**: 支持从联系人列表生成VCF，也支持从VCF文件解析联系人
- 🛡️ **安全防护**: 包含XSS防护措施，确保用户数据安全
- 📊 **仪表盘布局**: 清晰的三栏布局，操作流程一目了然

## 使用方法

### 输入格式

支持以下格式的联系人信息：

```
张三 13812345678
李四 13987654321
13712345678 王五
A0397 13612345679
```

### 操作步骤

1. 在左侧文本框中输入联系人信息（姓名和电话之间用空格或制表符分隔）
2. 点击"解析数据"按钮
3. 在中间面板查看联系人预览和统计信息
4. 点击"下载VCF"保存文件，或点击"复制"按钮复制VCF内容
5. 也可以使用"示例"按钮快速填充示例数据
6. 可以将VCF内容粘贴到右侧区域，点击"解析VCF"反向解析为联系人列表

## 安全与合规

### 数据安全
- 所有数据处理都在客户端本地完成，不会发送到任何服务器
- 实现了XSS防护，防止恶意脚本注入
- 对用户输入进行了验证和清理

### 隐私政策
- 不收集任何用户数据
- 不使用Cookie或追踪技术
- 不存储用户输入的信息

## 技术架构

项目采用模块化设计：

- `VCF生成.html`: 主页面，包含HTML结构和资源引用
- `css/style.css`: 样式文件，包含所有CSS规则
- `js/vcard-generator.js`: VCF生成核心逻辑
- `js/ui-controller.js`: UI交互控制逻辑

## 开发

项目完全基于原生JavaScript，无任何第三方依赖，适合前端初学者学习或作为模板使用。

### 模块说明

- [VCardGenerator](js/vcard-generator.js): 核心生成器类，负责解析联系人和生成VCF内容
- [UIController](js/ui-controller.js): UI控制器类，处理用户交互和DOM操作
